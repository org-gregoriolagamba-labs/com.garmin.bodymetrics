# BodyMetrics Documentation Hub

This directory contains the application documentation set for BodyMetrics.

## Language Model

- Italian and English are both first-class languages.
- Technical and functional documents should be maintained as mirrored files in `docs/it/` and `docs/en/`.
- The user guide may later gain an export-oriented source, but the repository remains the canonical source.

## Planned Structure

- `docs/it/` for Italian documentation.
- `docs/en/` for English documentation.
- `docs/shared/` for language-neutral assets such as diagrams, screenshots, and templates.

## Document Set

- Architecture overview.
- Functional specification.
- Technical reference.
- User guide.
- Troubleshooting and FAQ.
- Privacy and data handling.
- Build, validation, and release notes.
- Release notes.

## Entry Points

- Italian index: `docs/it/README.md`
- English index: `docs/en/README.md`
- Shared assets: `docs/shared/README.md`

## Maintenance Rules

- Keep the Italian and English versions aligned by document purpose and section order.
- Update architecture documentation before expanding dependent technical or user-facing documents.
- Mark manual-only verification steps explicitly when they cannot be automated in the repository.
