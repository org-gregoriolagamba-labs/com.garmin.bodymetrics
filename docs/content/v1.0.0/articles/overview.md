## What is BodyMetrics?

BodyMetrics is a **Garmin Connect IQ widget** for the Forerunner 265 that brings body composition tracking directly to your wrist.

Instead of opening a smartphone app, you can check your weight, body fat, muscle mass, hydration, and bone mass in seconds — along with derived values such as BMI, BMR, and power output — without leaving your watch.

## Why a Widget?

Garmin's **widget** type gives BodyMetrics a dedicated slot in the watch face loop, so it is always one swipe away. Unlike a full app, it launches instantly and does not interrupt any activity in progress.

## What Problems Does It Solve?

| Problem | BodyMetrics Solution |
|---------|----------------------|
| Body metrics scattered across phone apps | All values in one watch view |
| No history on the device | Local trend chart with selectable windows |
| No context for a reading | Clinical info screen with reference zones |
| Hard to remember personal targets | Target delta view per metric |
| Repeated manual re-entry | Garmin weight merged automatically |

## Supported Metrics

BodyMetrics tracks and displays **eight metrics**:

- **Weight** (kg) — manual entry or merged from Garmin UserProfile
- **Body Fat** (%) — manual entry
- **Muscle Mass** (kg) — manual entry; muscle percentage is derived
- **Hydration** (%) — manual entry
- **Bone Mass** (kg) — manual entry
- **BMI** — derived from weight and height
- **BMR** (kcal/day) — derived from profile data
- **Power** (W) — derived from muscle mass

## Supported Device

| Device | Garmin ID | API Level |
|--------|-----------|-----------|
| Forerunner 265 | `fr265` | ≥ 1.2.0 |

A **Lite build** targeting resource-constrained devices (FR55, FR735XT) is in planning.

## Languages

The interface is available in **English, Italian, French, and Spanish**. You can switch language at any time from the Preferences menu without losing any data.
