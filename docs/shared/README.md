# Shared Documentation Assets

This directory is reserved for language-neutral documentation assets.

## Intended Contents

- Diagrams used by both Italian and English documents.
- Screenshot assets for the user guide.
- Reusable templates and checklists.
- Export placeholders and publishing assets.
- Shared changelog and release-note templates.

## Rules

- Keep file names language-neutral when possible.
- Reference shared assets from both `docs/it/` and `docs/en/` instead of duplicating them.
