# Screenshot And Export Placeholders

This directory stores shared placeholders and final assets used by the bilingual user guides.

## Suggested Naming

- `setup-profile-step-1.png`
- `summary-main-screen.png`
- `trend-window-30d.png`
- `targets-editor-step.png`
- `reset-confirmation.png`

## Export Notes

- Keep a stable file name once a screenshot is referenced from both languages.
- Prefer one shared asset per screen state instead of duplicating by language when the UI is visually equivalent.
- If a screenshot must be language-specific, suffix it with `-it` or `-en`.
