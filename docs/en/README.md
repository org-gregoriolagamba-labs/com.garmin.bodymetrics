# BodyMetrics Documentation

This directory contains the English documentation for the BodyMetrics application.

## Documentation Set

- `architecture-overview.md`: architecture overview and layer boundaries.
- `functional-specification.md`: functional behavior and user workflows.
- `technical-reference.md`: technical contracts, storage, formulas, and integrations.
- `user-guide.md`: complete user guide.
- `troubleshooting.md`: known issues, diagnosis, and recovery guidance.
- `privacy-data-handling.md`: collected data, stored data, reset behavior, and privacy limits.
- `release-notes.md`: release notes and documented release contents.

## Recommended Reading Order

1. `architecture-overview.md`
2. `functional-specification.md`
3. `technical-reference.md`
4. `user-guide.md`
5. `troubleshooting.md`
6. `privacy-data-handling.md`
7. `release-notes.md`

## Maintenance Rules

- Every document must have a mirrored counterpart under `docs/it/`.
- Primary sections should stay in the same order in Italian and English.
- Technical details must be derived from code and should not be duplicated without an explicit owner.
